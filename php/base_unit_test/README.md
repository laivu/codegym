

# Unit Test - PHP - Base

> A base project for phpunit

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/laivu/codegym.git`

### Setup

> Move to folder and install by composer

```shell
$ cd php/base_unit_test
$ composer install
``` 

> Running demo

```shell
$ vendor/bin/phpunit
```
